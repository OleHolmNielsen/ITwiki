Note about sourced files:

The above table contains a general overview of the files that are sourced by the Bash shell in a plain vanilla environment. Which files are actually sourced, and in what order, depends on your personal setup. For example, many users like to call their ~/.bashrc from ~/.bash_profile. The SUSE Linux distribution has /etc/profile set up to call /etc/bash.bashrc and ~/.bashrc for all login shells, etc.

The file lists in the table were obtained from

  • the Bash info documents (`info bash`; also available here in HTML)
  • the Bash source code for version 3.0.16
